🛠️ Git身份配置指南

❗ 问题说明

Git需要配置用户名和邮箱才能提交代码，这是Git的安全机制。

🔧 一键解决方案

运行自动配置脚本：

📝 手动配置步骤

在终端中依次执行以下命令：

git config --global user.name "AnShiyi-1"
git config --global user.email "anshiyi-1@users.noreply.github.com"

✅ 验证配置

检查配置是否成功：

git config --global user.name git config --global user.email

🎯 配置完成后

现在你可以正常使用这些命令：

git add . git commit -m "你的提交信息" git push origin main

💡 小贴士

  • 用户名可以是任意字符串，不一定是GitHub用户名
  • 邮箱可以是GitHub提供的noreply邮箱
  • 配置只需做一次，全局生效
