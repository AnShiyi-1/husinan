🛠️ Git安装指南

⚠️ 检测到问题： 系统未找到Git命令

📥 步骤1：下载Git

Git for Windows官方下载：

下载Git for Windows 直接下载64位版本

⚙️ 步骤2：安装配置

安装时保持默认设置即可，确保勾选：

  • ✅ Git Bash
  • ✅ Git GUI
  • ✅ 添加到PATH环境变量

🔍 步骤3：验证安装

安装完成后，重新打开终端，运行：

git --version

🚀 步骤4：配置Git

设置用户名和邮箱：

git config --global user.name "你的名字"
git config --global user.email "你的邮箱@example.com"

💡 无需Git的替代方案

如果你不想安装Git，可以使用：

  • 🌐 GitHub网页界面上传文件
  • 📁 使用GitHub Desktop
  • 💻 使用VS Code内置Git
