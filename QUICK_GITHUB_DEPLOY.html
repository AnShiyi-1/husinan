🎨 一键部署到GitHub

1️⃣ 点击打开GitHub仓库

打开GitHub仓库

2️⃣ 上传文件

点击仓库页面的 "Upload files" 按钮

3️⃣ 拖拽上传

将以下文件拖拽到上传区域：

  • dist/index.html
  • dist/assets/ 文件夹
  • dist/*.svg 图标文件

4️⃣ 启用GitHub Pages

Settings → Pages → Deploy from a branch → main → Save
