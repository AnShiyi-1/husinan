🎨 情绪画布部署指南

📥 第一步：安装Git

访问 Git官网 下载并安装Git

🎯 第二步：一键部署

双击运行 upload-and-deploy.bat 文件

立即部署

⚠️ 注意事项

首次运行需要输入GitHub用户名和密码

✅ 完成部署

部署完成后，访问：
https://AnShiyi-1.github.io/qxhb

访问网站

🛠️ 备用方案

如果自动部署失败，请手动执行：

npm run build
git init
git add dist/ -f
git commit -m "Deploy"
git remote add origin https://github.com/AnShiyi-1/qxhb.git
git push -u origin main --force
