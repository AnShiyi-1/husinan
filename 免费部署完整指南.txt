## 📋 手动部署步骤（备用）

### 1. 准备工作
- 确保已安装 Node.js（https://nodejs.org）
- 确保已安装 Git（https://git-scm.com）

### 2. 安装Vercel CLI
```bash
npm install -g vercel
```

### 3. 部署命令
```bash
cd "c:\Users\Firday1\Desktop\绘画软件"
npm install
npm run build
vercel --prod
```

### 4. 配置步骤
当运行 `vercel --prod` 时：
- 选择 **GitHub** 登录
- 选择项目范围：**anshiyi-1's projects**
- 项目名称：使用默认或自定义
- 其他设置：保持默认即可

## 🛠️ 常见问题解决

### 问题1：部署失败
- 检查网络连接
- 确保package.json存在
- 重新运行一键部署脚本

### 问题2：页面空白
- 检查浏览器控制台
- 确认所有依赖已正确安装
- 重新构建项目

### 问题3：API无响应
- 检查server/index.js是否存在
- 确认环境变量配置正确
- 查看Vercel日志
